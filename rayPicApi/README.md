# 媒体图片相关 Api

## 描述

- 包含 `chooseImage`, `chooseMedia`, `chooseCropImage`, `getImageInfo`, `previewImage` 等相关 Api。

## 注意事项

- chooseCropImage 不支持自定义比例裁切。
